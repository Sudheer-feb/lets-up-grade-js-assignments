Methods in strings.

Primitive values, like "John Doe", cannot have properties or methods (because they are not objects).But with JavaScript, methods and properties are also available to primitive values, because JavaScript treats primitive values as objects when executing methods and properties.

1.searching a string in a string.

The search() method searches a string for a specified value and returns the position of the match.

2.extracting string parts.

There are three methods for extracting a part of a string.

2.1 The slice method.

The slice() Method extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: the start position, and the end position (end not included). example code : var str = "Apple, Banana, Kiwi"; var res = str.slice(7, 13); If a parameter is negative, the position is counted from the end of the string. example code: var str = "Apple, Banana, Kiwi"; var res = str.slice(-12, -6); If you omit the second parameter, the method will slice out the rest of the string example code : var res = str.slice(7);

2.2 The substring() Method.

substring() is similar to slice(). The difference is that substring() cannot accept negative indexes. example code : var str = "Apple, Banana, Kiwi"; var res = str.substring(7, 13);

2.3 The substr() Method

substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.

3. Replacing String Content

The replace() method replaces a specified value with another value in a string
